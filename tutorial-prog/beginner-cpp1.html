※Windows向けの説明です。 mac, linuxなどには対応していません。

またvisual studio前提での開発なのでご了承ください。

ITCではC++を教えるためにVisual Studioというソフトを使用しています。

無料で導入できるので、先駆けてプログラミングをしたい人はぜひダウンロードしてみてください。

このソフトはパソコンの性能がある程度整っている必要があります。

自分のパソコンでソフトを動かせるか確かめたい方は導入してみると良いと思います。

ダウンロード

Visual Studio 2019はこちらのサイトでダウンロードできます。

Visual Studio 2019のダウンロード

拡張機能

このページのコミュニティの下にある紫色の「無料ダウンロード」をクリックしてください。

別のページに移動し、しばらくするとインストーラが自動でダウンロードされます。

ダウンロードしたインストーラのexeファイルをダブルクリックし、起動してください。

起動してしばらく待つと次のような画面が出ると思います。

ここのC++によるデスクトップ開発の項目にチェックをし、右下のインストールのボタンをクリックしてください。(画像では「変更」となっています。)

するとインストールが開始されるので、しばらくお待ちください。(2,30分程度かかるときもあります。)

インストールが完了すると起動します。

ログインを求められるかもしれませんが、なくても大丈夫だと思います。

プロジェクトの作成

右にある「新しいプロジェクトの作成」をクリックし、

「空のプロジェクト」を選択し、右下の「次へ」をクリックします。

プロジェクト名はなんでもいいですが、ここではTestとします。

作成をクリックするとこのような画面が出てくると思います。基本的にはこの画面で作業していくことになります。

ソースコードを書くファイルを作る

右のソリューションエクスプローラーにあるソースファイルを右クリックし、新しい項目を選択します。

すると、ソースコードなどをプロジェクトに追加するためのウィンドウが開きます。

左のメニューからVisual C++を選択し、C++ファイル(.cpp)を選び、右下の「追加」をクリックします。

するとSource.cppというタブが表示されます。この中にプログラムを書いていくことになります！
